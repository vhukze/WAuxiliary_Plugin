# 🎙️ TIAX文转音插件使用说明

## 完全免费内置无限余额的KEY

## 快速指令

| 指令 | 说明 |
|------|------|
| `音色列表` / `帮助` / `说明` | 查看 321 种音色列表 |
| `音色 7` | 修改默认音色为 7 号 |
| `#tts 你好` | 用默认音色生成语音 |
| `#tts 38 你好` | 临时用 38 号音色（丁真）生成语音 |

## 音色范围
1～321 号，发送 `音色列表` 可查看完整清单。

## 默认音色
初始为 7 号（爱莉），可通过指令或设置修改。

## 注意事项
- 需要联网，生成需几秒
- 自动转换为微信支持的 SILK 格式
- 临时音色不影响默认设置

**使用方法**：在聊天框直接发送上述指令即可。
