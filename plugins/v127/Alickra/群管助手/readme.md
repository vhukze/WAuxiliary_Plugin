# 🛡️ 群管助手

> 轻量高效的微信群管理插件，社区版免费使用，完整版邀请码激活。

---

## ✨ 功能一览

### 🆓 社区版（开箱即用）

| 功能 | 说明 |
|:-----|:-----|
| 🎉 入群欢迎 | 自动发送欢迎词，支持 `{昵称}` 变量 |
| 🚫 违禁词过滤 | 检测违禁内容自动提醒，支持动态添加 |
| 💬 关键词回复 | 自定义关键词触发自动回复 |
| 📋 命令管理 | `#帮助` 查看所有可用命令 |

### 🔑 完整版（邀请码激活）

| 功能 | 说明 |
|:-----|:-----|
| ✅ 签到系统 | 每日签到 + 连续签到加分 |
| 🏆 积分排行 | 积分榜 / 活跃度榜 |
| 👢 踢人拉黑 | `#踢 @xxx` / `#拉黑 @xxx` |
| 👑 权限管理 | 设置/移除群管理员 |
| 🤝 自动验证 | 好友请求自动通过 |
| 💓 心跳保活 | 后台稳定运行 |

---

## 📦 安装

1. 将插件文件夹复制到 WAuxiliary 插件目录
2. 重启 WAuxiliary 或重新加载插件
3. 在群里发送 `#帮助` 验证安装成功

## ⚙️ 配置

首次使用请在 **插件设置** 中配置：

```
super_admins    → 管理员微信ID（JSON数组，如 ["wxid_xxx"]）
server_url      → 服务器地址（激活完整版需要）
```

## 📖 命令列表

### 所有人可用

| 命令 | 说明 |
|:-----|:-----|
| `#帮助` | 显示帮助信息 |
| `#签到` | 每日签到（完整版） |
| `#积分` | 查看我的积分（完整版） |
| `#排行` | 积分排行榜（完整版） |

### 管理员命令

| 命令 | 说明 |
|:-----|:-----|
| `#设置欢迎词 内容` | 设置入群欢迎词，支持 `{昵称}` |
| `#添加违禁词 词1 词2` | 添加一个或多个违禁词 |
| `#删除违禁词 词` | 删除指定违禁词 |
| `#违禁词列表` | 查看当前所有违禁词 |
| `#添加回复 关键词=回复内容` | 添加关键词自动回复 |
| `#删除回复 关键词` | 删除指定回复规则 |
| `#回复列表` | 查看所有回复规则 |
| `#踢 @xxx` | 踢出群成员（完整版） |
| `#拉黑 wxid` | 拉黑指定用户（完整版） |

---

## 🔓 激活完整版

在群里发送：

```
#激活 你的邀请码
```

激活全自动完成，包括下载、校验、解密、加载，无需手动操作。重启后自动加载完整功能。
