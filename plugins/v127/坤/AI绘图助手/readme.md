# AI绘图助手

### 功能介绍
本脚本通过对接 OpenAI 格式的或自定义 API 接口，实现微信聊天内输入指令自动生成图片并回复。

### 使用方法
1. **配置 Key**: 请在 `main.java` 的 `你的apiKey` 变量中填写你的 API Key。
2. **触发指令**: 在聊天窗口发送：`/绘图 [描述词]`。
   - 示例：`/绘图 一只戴着墨镜的熊猫在沙滩喝椰子水`
3. **响应**: 脚本会调用接口生成图片，下载到缓存目录后自动发送回当前聊天。

### 注意事项
- API 响应速度取决于网络和绘图模型复杂度，请耐心等待 `toast` 提示。
