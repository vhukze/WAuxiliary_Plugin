## 📖 脚本概述

这是一个朋友圈自动点赞脚本，可在后台刷新并捕捉近期朋友圈动态，根据你设置的好友规则、时间范围、延迟策略和内容过滤条件自动执行点赞。

## 🚀 快速开始

点击插件设置或在微信聊天窗口输入以下任一指令并发送，即可打开设置界面：

- `自动点赞设置`
- `点赞设置`

## 🔧 主要功能

- 启用/关闭自动点赞：可随时总开关控制。
- 点赞对象规则：
  - 白名单模式：只点赞选中的好友。
  - 黑名单模式：点赞除黑名单外的所有好友。
- 延迟点赞策略：
  - 固定延迟点赞。
  - 随机延迟点赞，降低行为规律性。
- 后台刷新策略：
  - 可设置刷新间隔。
  - 支持全天循环或指定时间段循环。
- 安全限制：
  - 仅处理指定小时数内的近期朋友圈。
  - 可设置自动点赞运行时段。
  - 可配置无法识别时间/类型时的处理策略。
- 内容过滤：
  - 可跳过文字、图文、视频等类型。
  - 支持关键词过滤，并可指定关键词作用于文字/图文/视频。
- 朋友圈发布通知：
  - 可对指定好友的新朋友圈发布通知或 Toast 提醒。
- 日志记录：
  - 可记录成功、跳过、异常等运行日志，便于排查问题。

## 🧩 配置入口

- 主入口：任意输入框输入`自动点赞设置` 或 `点赞设置`点击发送，或点击插件设置
- 点赞对象规则：配置白名单/黑名单好友。
- 延迟点赞策略：配置固定或随机延迟。
- 后台刷新策略：配置刷新开关、刷新间隔和刷新时间段。
- 安全限制：配置朋友圈时效和运行时段。
- 内容过滤：配置跳过类型和关键词过滤。
- 朋友圈发布通知：配置指定好友朋友圈发布提醒。
- 日志记录：查看或控制日志记录。

## 💡 使用建议

- 首次使用建议先选择“白名单模式”，只添加少量好友测试。
- 不建议黑名单为空时直接启用黑名单模式，否则会对所有捕捉到的好友朋友圈自动点赞。
- 延迟策略建议使用随机延迟，避免点赞行为过于集中。
- 后台刷新间隔不宜设置过短，以减少耗电和异常风险。
- 如果发现没有点赞，先检查是否启用插件、好友是否在规则内、朋友圈是否超过时效或被内容过滤命中。

## ⚠️ 注意事项

- 插件仅处理后台刷新捕捉到的近期朋友圈，不保证所有朋友圈都能被捕捉到。
- 不同微信版本、WAuxiliary版本和系统环境可能影响点赞成功率。
- 请合理使用自动点赞功能，避免频率过高导致账号风险。
- 修改配置后如未立即生效，可尝试重载插件或重启微信。
